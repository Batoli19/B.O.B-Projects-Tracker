# Bank of Botswana — Project Tracker (BoB Tracker)

Professional internal Project Tracker web application for the Bank of Botswana.

**Stack:** React + Vite + Tailwind CSS + React Router + Recharts + Lucide React

## Local Development

```bash
cd bob-project-tracker
npm install
npm run dev
```
